Boson Framework File Response Example

This is a static HTML file served by the Boson Framework's StaticFiles middleware.

Visit API Demo Home to see all the file response examples.

This example demonstrates how to:

  • Send files with proper MIME type detection
  • Force file downloads with Content-Disposition
  • Stream large files efficiently
  • Use ETags and Last-Modified for caching
  • Implement range requests for partial content

The API is based on Express.js-like patterns but implemented in modern C++.
